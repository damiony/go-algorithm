# 数组

### 语法

> 数组是一个由固定长度的特定类型元素组成的序列。
>
> 每个元素通过索引下标访问，索引下标范围从0开始到数组长度减1的位置。
>
> 内置的`len`函数返回数组中元素的个数。
>
> 数组长度是数组类型的一部分，因此`[3]int`和`[4]int`是两种不同的数组类型。数组长度必须是常量表达式。



### 创建

- 直接声明：

  ```go
  var a [3]int
  ```

  数组中的每个元素都会被初始化为零值。

- 使用字面值语法，顺序初始化：

  ```go
  var a [3]int = [3]int{1, 2, 3}
  ```

  或者：

  ```go
  a := [...]int{1, 2, 3}
  ```

- 使用索引值的方式初始化：

  ```go
  a := [...]int{100: -1}
  ```



### 比较

如果数组的元素类型可以比较，则数组类型也可以比较。

只有两个数组的所有元素都相等，数组才是相等的。
